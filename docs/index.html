Music Metrics: An Interactive Dashboard

Team Member: Ahmed Bilal Naizm (Reg. No. 2022064)

Project Description

Music Metrics is a sophisticated, interactive web-based dashboard designed for the deep exploration of musical trends from 1900 to 2021. Moving beyond static charts, this project provides a dynamic and engaging user experience, allowing users to filter, dissect, and visualize a massive dataset of over 586,000 songs. The dashboard reveals compelling narratives in music history, from the evolution of danceability to the sonic characteristics of hit songs, all through a sleek, modern interface.

Dataset Description

The analysis is powered by the comprehensive "Spotify Tracks DB" from Kaggle, which includes two primary files:

  • tracks.csv: Contains detailed audio features for 586,672 songs, including energy, danceability, valence, tempo, and popularity.
  • artists.csv: Provides genre information for over 1.1 million artists, which is used to classify songs and analyze genre-specific trends.

This rich dataset enables a granular look at the defining attributes of music across different eras and genres.

Tools and Technologies

  • HTML5: For the core structure and content of the web application.
  • CSS3: For all styling, including the dark theme, responsive design, and custom animations.
  • JavaScript (ES6+): For data processing, interactivity, and dynamic chart generation.
  • Plotly.js: A powerful charting library used to create all 7 interactive visualizations.
  • Git & GitHub Pages: For version control and hosting the live, publicly accessible dashboard.
