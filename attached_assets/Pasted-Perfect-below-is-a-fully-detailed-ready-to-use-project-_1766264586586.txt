## 1. Project Overview

**Music Trends Time Machine** is an interactive data analytics project that explores how musical characteristics have evolved over time using large-scale Spotify data. The project provides a dynamic dashboard that allows users to analyze temporal trends in music, compare genres and eras, simulate recommendation strategies, detect anomalous shifts in sound, and follow a guided narrative of music evolution.

Unlike static visualizations, this project is designed as a **product-like analytics application**, combining exploratory data analysis with interactive simulation and automated insight generation. The dashboard enables users to engage with music data intuitively, uncover patterns across decades, and understand how audio features such as energy, danceability, tempo, and valence have changed over time.

## 2. Objectives of the Project

The main objectives of this project are:

* To analyze **long-term temporal trends** in musical audio features.
* To provide **interactive visualizations** that allow users to filter by year, decade, and genre.
* To simulate **‚Äúwhat-if‚Äù scenarios** by allowing users to assign custom weights to musical features.

## 5. Stand-Out ‚ÄúExtra Ingredient‚Äù Features

### 5.1 Interactive What-If Simulator

This feature allows users to simulate music recommendation strategies by adjusting sliders that represent the importance (weights) of different audio features.

**Example use case:**

* Increase the weight of energy and tempo to see how upbeat music trends dominate certain decades.
* Observe how genre rankings change when different features are prioritized.

The simulator recalculates a custom score in real time and updates charts and rankings accordingly.
